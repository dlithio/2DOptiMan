2DOptiMan

What does this software do?

This project contains the files necessary to compute the stable or unstable manifold of a vector field using MATLAB, and then create a vtk file to be read by Paraview for visualization. Specifically, we consider the system \( \frac{du}{dt} = F(u(t)) \), where \( u(t)=(x(t),y(t),z(t)) \) with a fixed point where \( F(x_0, y_0 , z_0 ) = 0 \). In the case of a stable manifold, we find the set of points such that as \( t \rightarrow \infty \), \( u(t) -> (x_0,y_0,z_0) \). In the case of the unstable manifold, we find the set of points such that as \( t \rightarrow -\infty \), \( u(t) -> (x_0,y_0,z_0) \).

How do you get this to work?

Here we'll guide you through what you need to get this running, and how to visualize your first manifold.

The Prerequisites

You'll need this software before you can get up and running.

MATLAB

For the computations, you'll need a copy of MATLAB.. While this software costs money, you may be able to use the trial version, buy the student one, or even use a copy on college or university machines.

ParaView

For visualization, you'll need to download and install ParaView. Following the previous link will help you install it for Windows and Mac, and for Debian and Ubuntu you can use

$ sudo apt-get update
$ sudo apt-get install paraview

Many other linux distributions have ParaView in the default repositories, so installation should be just as simple on those machines as well.

This Software

Click one of the download links on the left side of this page and put the archive (the zip or the tar file) in a place you can find. You'll need to unzip (or untar) it. In Windows/Mac/Linux, this is usually a matter find finding the file, double clicking it, and finding some unzip (or untar) option.
